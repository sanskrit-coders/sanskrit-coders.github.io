### Safety
- Can cut off fingers and draw blood.
- Move the handle to vertical position, lock it and remove the handle, whenever not in use.
- Lower the page holder to cover the blade from one side by turning the knob provided.


## Parts

Visible from the board side -

-   Moveable file (to set papers on the board).
-   Cutter lock (To hold the blade handle up close to 90 degrees)
-   Base (on which paper is placed)
-  (Paper) Pressing handle (roated to lower the pressing foot)
-   Cutter handle (detachable)

Visible from the cutting side

-   Plastic file ( cutting strip or cutting mat of the paper cutter)
-   Blade splint (lid of the "movable file" above)

2.  Please take out the file circumvolution for 90 deg angle and return to the embed-slot again by pressing. When it is difficult to cut paper completely. The file can be using for 8 directions. After using one direction with 4 sides, you can extract out the file using on the other sides.
